# מערכת מלאי — בית דפוס 🖨️

אפליקציית React לניהול מלאי של בית דפוס: מוצרים, כמויות, מידות חולצות, רוחבי גליל לשמשוניות, ספקים עם וואטסאפ, תמונות מוצרים, ערכות צבע ועוד.

## תכונות

- ✅ ניהול כמויות עם כפתורי +/− והקלדה ישירה
- 👕 ניהול לפי מידות לחולצות (2–18, S–4XL)
- 📏 ניהול לפי רוחב גליל לשמשוניות (105/127/158 ס"מ)
- 🚚 רשימת ספקים לכל מוצר עם שליחת הודעת וואטסאפ בלחיצה
- 📸 תמונה לכל מוצר + תצוגה מוגדלת בריחוף
- 🔴🟠 צביעת שורות לפי מצב מלאי (אזל / נמוך) + צליל אזהרה
- 🏢 פרטי חברה (שם, ח.פ, כתובת) ולוגו בלחיצה
- 🎨 6 ערכות צבע לממשק
- 💾 שמירה אוטומטית (localStorage בדפדפן)

## הרצה מקומית

```bash
npm install
npm run dev
```

ואז פותחים את הכתובת שמופיעה בטרמינל (בדרך כלל http://localhost:5173).

## העלאה ל-GitHub

1. צרו ריפוזיטורי חדש ב-github.com (כפתור **New repository**), למשל בשם `print-shop-inventory`. אל תסמנו יצירת README.
2. בתיקיית הפרויקט הריצו:

```bash
git init
git add .
git commit -m "מערכת מלאי לבית דפוס - גרסה ראשונה"
git branch -M main
git remote add origin https://github.com/<שם-המשתמש-שלכם>/print-shop-inventory.git
git push -u origin main
```

3. זהו! הפרויקט באוויר. 🎉

## פריסה לאינטרנט (אופציונלי)

אפשר לפרוס בחינם ב-Vercel או Netlify — מתחברים עם חשבון GitHub, בוחרים את הריפו, והם מזהים Vite אוטומטית. פקודת build: `npm run build`, תיקיית פלט: `dist`.
